Функсияҳои сатр

Якчанд роҳҳо барои идора кардани сатр бо истифода аз функсияҳои дарунсохташудаи PHP мавҷуданд. Дар ин мақола, мо 15 функсияҳои маъмултарини сатри PHP-ро меомӯзем:

  1. strlen()
  2. strpos()
  3. str_replace()
  4. ucwords()
  5. strtoupper()
  6. strtolower()
  7. substr()
  8. trim()
  9. explode()
  10. implode()

1. strlen($str) - ин функсия дарозии сатр ё шумораи аломатҳоро дар сатр, аз ҷумла фосилаҳои холӣ бармегардонад.

Код:

$str = "Welcome to Kodnavis";

// истифодаи strlen дар усули echo
echo "Length of the string is: ". strlen($str);

Натиҷа:

Length of the string is: 23

2. strpos($str, $text) - ин функсия барои дарёфти мавқеи матн/калима дар сатри додашуда истифода мешавад. Ҳар як ҳарфи сатр дорои индекс мебошанд, яъне дар индекси 0 аввалин ҳарфи сатр ҷойгир мебошад, дар индекси 1 ҳарфи дуюми сатр ва ғ. Мисли массив, индекси сатр низ аз сифр шуруъ мешавад.

Код:

$str = "Welcome to Kodnavis";

// истифодаи strpos дар усули echo
echo "Position of 'Kodnavis' in string: ". strpos($str, 'Kodnavis');

Натиҷа:

Position of 'Kodnavis' in string: 11

3. str_replace($replacethis, $replacewith, $str) - ин функсия барои иваз кардани як қисми сатр бо баъзе матн истифода мешавад. Ҳангоми истифодаи ин функсия, аргументи аввал қисми сатрест, ки шумо онро иваз кардан мехоҳед, аргументи дуюм матни навест, ки шумо мехоҳед дохил кунед ва аргументи охирин худи тағирёбандаи сатр аст.

Код:

$str = str_replace("Kodnavis", "Kodnavis.ru", "Welcome to Kodnavis");

echo $str;

Натиҷа:

Welcome to Kodnavis.ru

4. ucwords($str) - ин функсия барои формат кардани сатр истифода мешавад. Ин функсия аввалин ҳарф/аломати ҳар як калимаро дар сатр ба ҳарфи калон табдил медиҳад.

Код:

$str = "welcome to kodnavis";
 
echo ucwords($str);

Натиҷа:

Welcome To Kodnavis

5. strtoupper($str) - ин функсия барои табдил додани ҳар ҳарф/аломати ҳар як калимаи сатр ба ҳарфи калон истифода мешавад.

Код:

$str = "welcome to kodnavis";

echo strtoupper($str);

Натиҷа:

WELCOME TO KODNAVIS

6. strtolower($str) - ин функсия барои табдил додани ҳар ҳарф/аломати ҳар як калимаи сатр ба ҳарфи хурд истифода мешавад.

Код:

$str = "WELCOME TO KODNAVIS";
                
echo strtolower($str);

Натиҷа:

welcome to kodnavis

7. substr($str, $start, $length) - ин функсия барои истихроҷи қисми сатр (зерсатр), ки аз мавқеи муайян, дарозии муайян оғоз мешавад, истифода мешавад.

Аргументи аввал худи сатр аст, аргументи дуюм индекси ибтидоии зерсатри истихроҷшаванда ва аргументи сеюм дарозии зерсатри истихроҷшаванда мебошад.

Код:

$str = "Welcome to Kodnavis";
                
echo substr($str, 11, 8);

Натиҷа:

Kodnavis

8. trim($str, charlist) - ин хусусият барои нест кардани ҷойҳои зиедатӣ дар аввал ва охири сатр истифода мешавад. Аргументи дуюми charlist ихтиерӣ аст. Мо метавонем рӯйхати аломатҳоро ба мисли сатр ҳамчун аргументи дуюм барои нест кардани ин аломатҳо аз сатри асосӣ пешниҳод кунем.

Код:

$str1 = " Hello World ";
echo trim($str1);

$str2 = "Hello Hello";
echo trim($str2,"Heo");

Натиҷа:

Hello World
llo Hell

Тавре, ки шумо дар натиҷа мебинед, дар ҳолати аввал, ҷойҳои иловагӣ дар аввал ва охири сатр нест карда шудаанд ва дар ҳолати дуюм, аломатҳои зикршуда аз аввал ва охири сатр хориҷ карда шудаанд.

9. explode(separator, $str, $limit) - ин функсия барои тақсим кардани сатр, сохтани массив аз қисмҳои шикастаи сатр ва баргардонидани массив истифода мешавад. Аргументи аввал, separator, муайян мекунад, ки сатрро аз куҷо бояд шикаст. Ин метавонад фосила, дефис(-) ё ягон аломати дигар бошад

Аргументи дуюм худи сатр аст ва аргументи сеюм маҳдудиятест, ки шумораи элементҳои баргардонидашудаи массивро муайян мекунад. Аргументи сеюм ихтиёрӣ аст.

Код:

$str = "Its a beautiful day";
                
// we can directly print the result of explode
print_r(explode(" ", $str));

Натиҷа:

Array (
    [0] => Its
    [1] => a
    [2] => beautiful
    [3] => day
)

Дар мисоли боло, мо фосиларо ҳамчун тақсимкунанда барои шикастани сатр ва баргардонидани массив нишон додем.

Агар мо инчунин limit аргументи сеюмро муайян кунем, мо метавонем шумораи элементҳои баргардонидашудаи массивро маҳдуд кунем. Масалан, агар мо 2-ро ҳамчун аргументи сеюм нишон диҳем, пас мо танҳо 2 элементро дар массив мегирем, дутои аввалро.

10. implode(separator, $arr) - ин функсия барои сохтани сатр бо истифода аз элементҳои массив аз массиви додашуда ва якҷоя кардани онҳо бо истифода аз тақсимкунанда истифода мешавад.

Код:

$arr = array("Its", "a", "beautiful", "day");

echo implode(" ", $arr);
echo implode("-", $arr);
echo implode("/", $arr);

Натиҷа:

Its a beautiful day
Its-a-beautiful-day
Its/a/beautiful/day
