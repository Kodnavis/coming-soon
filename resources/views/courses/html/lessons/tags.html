Тегҳои HTML чист?

HTML тегҳо фармонҳои махсус барои браузер мебошанд. Онҳо ба ӯ мегӯянд, ки масалан, чӣ гуна, бояд сарлавҳаи саҳифа нишон дода шавад ва ё сархатҳо.

Тегҳо тибқи ин принсип сохта мешаванд: аломати хурд <, баъдан номи тег ва сипас аломати калон >, ба монанди ин: <номи теги>. Номи тегҳо метавонад аз ҳарфҳо ва рақамҳои англисӣ иборат бошад. Намунаҳои тегҳо: <h1>, <p>, <b>.

Тегҳо одатан ҷуфт навишта мешаванд - теги кушодашави ва теги пӯшидашави. Фарқи байни тегҳои кушодашави ва пӯшидашави дар он аст, ки теги пӯшидашави аломати тақсим / дорад пас аз аломати хурд <.

Масалан, <p> - теги p-ро кушодем ва ҳамин тавр - </p> - онро бастем. Ҳар чизе, ки дар байни тегҳои боз ва баста бошад таҳти таъсири теги мо қарор мегирад.

Ҳастанд тегҳое, ки ба пӯшида шудан ниёз надоранд, ба монанди <br> ё <img>.
