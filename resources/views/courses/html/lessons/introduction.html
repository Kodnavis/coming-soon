Муқаддима ба HTML

Барои сохтани вебсайт, шумо бояд бисёр забонҳои гуногуни вебро донед.

Забонҳои HTML ва CSS барои тарҳрезии сайт пешбинӣ шудаанд (тарҳбандӣ - ҷойгиркунии унсурҳои сайт дар ҷойҳои мувофиқ). Забони PHP барои барномасозии сайт лозим аст (бо ёрии он, масалан, шумо метавонед корбаронро сабти ном кунед). Забони JavaScript барои "аниматсия" кардани сайт лозим аст: масалан, тасвирҳои тағирёбанда (слайдер).

HTML асоси вебсайтҳо мебошад ва он чорчӯбаи саҳифаро, ки шумо дар браузер мебинед, месозад. Агар шумо як сафҳаи сайтро бо як саҳифаи китоби коғази муқоиса кунед, пас сайт, ба монанди китоб, сархатҳо ва сарлавҳаҳо дорад.

Китоб ба худ ном (дар асл, сарлавҳаи асоси), сарлавҳаи бобҳо ва параграфҳо дар бар мегирад. Сарлавҳаҳо, параграфҳо ва инчунин дигар блокҳоро дар саҳифаи сайт метавон қайд кард. Ин бо истифода аз тегҳои HTML анҷом дода мешавад.
